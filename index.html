Major Features in PHP 8.5

New

Property hooks

Define getter and setter logic directly on properties with a clean, intuitive syntax. Reduce boilerplate while maintaining encapsulation.

Learn more

example.php 

    class User {
        public string $name {
            set => ucfirst($value);
        }
        
        public string $email {
            set {
                if (!filter_var($value, FILTER_VALIDATE_EMAIL)) {
                    throw new ValueError('Invalid email');
                }
                $this->email = strtolower($value);
            }
        }
    }

example.php 

    class Post {
        // Public read, private write
        public private(set) string $id;
        public private(set) DateTime $createdAt;
        
        public function __construct() {
            $this->id = uniqid();
            $this->createdAt = new DateTime();
        }
    }
New

Asymmetric visibility

Control read and write access independently. Make properties publicly readable but privately writable without verbose getter methods.

Learn more
Enhanced

New array functions

Powerful array manipulation utilities that make common operations more expressive and reduce the need for custom implementations.

Learn more

example.php 

    // array_find - get first matching element
    $user = array_find($users, fn($u) => $u->id === 42);

    // array_find_key - get first matching key
    $key = array_find_key($data, fn($v) => $v > 100);

    // array_any / array_all - existence checks
    $hasAdmin = array_any($users, fn($u) => $u->isAdmin);
    $allActive = array_all($users, fn($u) => $u->active);
