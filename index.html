Property hooks

Define getter and setter logic directly on properties with a clean, intuitive syntax. Reduce boilerplate while maintaining encapsulation.

Learn more

example.php 

    class User {
        public string $name {
            set => ucfirst($value);
        }
        
        public string $email {
            set {
                if (!filter_var($value, FILTER_VALIDATE_EMAIL)) {
                    throw new ValueError('Invalid email');
                }
                $this->email = strtolower($value);
            }
        }
    }

example.php 

    class Post {
        // Public read, private write
        public private(set) string $id;
        public private(set) DateTime $createdAt;
        
        public function __construct() {
            $this->id = uniqid();
            $this->createdAt = new DateTime();
        }
    }

Property hooks

Eliminate boilerplate getter/setter methods with property hooks

example.php 

    class User {
        private string $name;
        
        public function getName(): string {
            return $this->name;
        }
        
        public function setName(string $name): void {
            $this->name = ucfirst($name);
        }
    }

    $user = new User();
    $user->setName('john');
    echo $user->getName(); // John

example.php
